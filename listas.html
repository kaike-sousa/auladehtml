Desafios de lista

Crie um título principal para a página, tipo: "Time dos Sonhos dos Animes". Faça uma lista ordenada com 5 personagens de animes que você escolheria para montar um time (pode ser por ordem de poder, preferência etc.). Depois, crie uma lista não ordenada com os itens que esse time levaria para a batalha (ex: kunai, esferas do dragão, ramen... use a criatividade!). Por fim, faça uma lista de descrição explicando por que escolheu 3 desses personagens. Cada personagem deve ter uma explicação engraçada, épica ou criativa. Use ao menos dois atributos type diferentes nas listas ordenadas (ex: type="A", type="1").
